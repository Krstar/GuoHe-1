## 果核APP
专注于为江苏科技大学学生提供更好的校园、生活、个性服务。
[传送门--->果核.](http://106.14.220.63/)

## 功能
####校园服务（无需江科大vpn导入）
>* 课表查询 
>* 成绩绩点
>* 图书馆藏
>* 校车查询
> * 空教室查询
> * 俱乐部查询
> * 早操查询
> * 就业信息（正在集成）

#### 生活服务
>*  纪念日
>* 待办事项 
>* 快递查询
>* 文字识别
>* 在线翻译
>* 分贝计
>* 表情包制作

#### 个性推荐
>* 短篇小说
>* 每日趣闻
>* 星座运势
> * 精选段子
> * 历史上的今天
